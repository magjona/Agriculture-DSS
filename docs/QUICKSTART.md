# QUICK START CHEAT SHEET

## 🚀 Get Started Now

### Run These Commands in a Fresh Terminal!
```powershell
cd C:\Users\MAGADA\Desktop\MJR\vscode
venv\Scripts\activate
python manage.py makemigrations
python manage.py migrate
python manage.py seed_crops
python manage.py runserver
```

Then open: **http://127.0.0.1:8000**

## 👩🌾 For Farmers

| What You Can Do | How to Do It |
|------------------|--------------|
| Create an Account | Go to Sign Up |
| Login | Use username OR farm name |
| Add a Farm | Dashboard → Add Farm |
| Get Recommendations | Click "Generate Recommendations" for a farm |
| Edit Profile | Menu → Edit Profile |

## 📊 For Managers/Admins
